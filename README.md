# Project W.A.V.E.S.

This is my submission for the GameOff 2025 jam. It's a topdown shooter made with pygame.

---

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/mbonitho/GameOff2025.git
cd GameOff2025
```
### 2. Create a virtual environment
```bash
python3 -m venv .venv
```
### 3. Activate the virtual environment
* Linux/macOS
```bash
source venv/bin/activate
```
* Windows (powershell)
```bash
venv\Scripts\Activate.ps1
```
### 4. Install dependencies
```bash
pip install -r requirements.txt
```
### 5. Run the game
```bash
python main.py
```
### Notes
* Make sure you have Python 3.8+ installed.
* If you encounter issues, try upgrading pip:
```bash
pip install --upgrade pip
```
